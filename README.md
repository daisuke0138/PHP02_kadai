## ②課題内容（どんな作品か）

①アカウント作成画面：社員番号、氏名の入力とloginpassの作成。
データベースに社員番号、氏名、passを保存

②loging画面で社員番号とpassを入力。
データベースと照合し、loginする。

③menu画面はあるが、logoutボタンのみ
ボタンクリックでtopに戻ります。
